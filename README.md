# Uber Data Analysis (SQL + Power BI)
End-to-end Data Analysis project using SQL & Power BI

## project Overview

This project analyzes Uber ride booking data to uncover insights related to revenue, cancellations, and customer behavior using SQL and Power BI.

---

## Tools & Technologies

* SQL (MySQL)
* Power BI
* Excel

---

## Dataset

* 20,000+ ride records
* Includes booking status, vehicle type, revenue, distance, and ratings

## Key Insights

* Premium vehicle types generate higher revenue
* Cancellation rate is around **25%**, impacting overall revenue
* Driver-related issues are the major cause of cancellations
* High booking volume but significant revenue loss due to cancellations

---

## Business Recommendations

* Improve driver accountability to reduce cancellations
* Promote premium vehicle usage to increase revenue
* Optimize ride allocation and customer experience
